When I started this research, I had a choice: publish a paper pointing out that LLMs produce miscalibrated advice, or build a system that fixes it. I chose both.

CheckMyCoach isn't just a paper contribution — it's 44 passing tests, a functioning API pipeline, a Streamlit frontend, and a cost per correction of ~$0.001.

Why does this matter? Because **trust calibration research needs to be deployable.** If we find that AI advice is systematically misleading but only propose theoretical fixes, we've done half the job. The other half is building things that work.

This philosophy extends to reproducibility. Every project on this site includes:
- A quick-start guide (3 steps to run)
- Full environment specification
- Tested code with CI

Not because it's required for publication, but because **work you can't reproduce is work that doesn't exist.**
